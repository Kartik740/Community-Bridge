Make changes in VS Code
       ↓
npm run build        (only if you changed the React web app)
       ↓
git add .
git commit -m "what you changed"
git push             (updates both GitHub and triggers Render auto-deploy)
       ↓
firebase deploy --only hosting    (only if you changed the React web app)

To redeploy the app-

Step 1 — Build new APK
cd /d/CommunityBridge/mobile
flutter build apk --release

Step 2 — Update GitHub Release

Go to your GitHub repo → Releases → click edit (pencil icon) on v1.0.0
Delete the old app-release.apk
Upload the new app-release.apk
Click Update release
